# Perguntas Frequentes

<Disclosure>
  <Disclosure.Button>💳 Como funciona o pagamento pelo sistema?</Disclosure.Button>
  <Disclosure.Panel>
    Os pagamentos são processados automaticamente através das faturas geradas pelo sistema. Você pode personalizar os métodos aceitos nas configurações.
  </Disclosure.Panel>
</Disclosure>

<Disclosure>
  <Disclosure.Button>📦 Como criar um envio para o cliente?</Disclosure.Button>
  <Disclosure.Panel>
    Basta acessar o módulo "Envios", clicar em "Novo Envio" e selecionar os produtos e cliente desejado. O sistema vai gerar a fatura e rastreio automaticamente.
  </Disclosure.Panel>
</Disclosure>

<Disclosure>
  <Disclosure.Button>📁 Posso adicionar arquivos ou fotos nas faturas?</Disclosure.Button>
  <Disclosure.Panel>
    Sim. Dentro de cada fatura, há a opção de anexar imagens ou documentos para o cliente visualizar.
  </Disclosure.Panel>
</Disclosure>
